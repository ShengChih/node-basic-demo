node-basic-demo
==================

## 基於 Exporessjs 實作簡易 CRUD 功能（貨品管理）

- Express MVC
- CRUD JSON API

## Requirments

- MongoDB
- NodeJS

## Debug Tool

- Postman
- Chrome Inspect with NodeJS inspect

## Use
- [express](https://github.com/expressjs/express)
- [mongoose](https://www.npmjs.com/package/mongoose)
- [nodemon](https://www.npmjs.com/package/nodemon)
